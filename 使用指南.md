# Agnes Video 技能 - 快速使用指南

## 🎯 快速开始

### 第一步：获取 API Key

1. 访问 **https://agnes-ai.com**
2. 注册账号并登录
3. 在个人中心申请 API Key
4. 保存 API Key：
   ```bash
   agnes key set "您的_API_KEY"
   ```

### 第二步：生成第一个视频

```bash
cd ~/.trae/skills/agnes-video
python video_manager.py generate --prompt "一只猫在海滩上散步" --num-frames 121 --frame-rate 24
```

### 第三步：查看结果

视频生成完成后，会返回完整的 JSON 信息，包含视频链接。

## 📖 常用命令

### 生成视频

```bash
# 基本用法（5 秒视频）
python video_manager.py generate --prompt "您的视频描述"

# 自定义参数
python video_manager.py generate \
  --prompt "日落时分的海边风景" \
  --num-frames 241 \    # 10 秒视频
  --frame-rate 24

# 关键帧动画
python video_manager.py generate \
  --prompt "从白天过渡到夜晚" \
  --image ./day.png \
  --image ./night.png \
  --mode keyframes
```

### 查询状态

```bash
# 单次查询
python video_manager.py status <video_id>

# 持续查询直到完成
python video_manager.py status <video_id> --poll --interval 5

# 查询 60 次（最多 5 分钟）
python video_manager.py status <video_id> --poll --interval 5 --max-attempts 60
```

### 检查配置

```bash
# 检查系统状态
python video_manager.py check

# 引导 API Key 配置
python video_manager.py guide
```

## 📋 参数详解

### --prompt（必需）

**视频描述**，详细具体的描述有助于生成更好的视频。

✅ **好的描述**：
- "一个穿着红色连衣裙的女孩在海边跳舞，夕阳西下，金色光线洒在海面上"
- "卡通风格的猫咪在草地上追逐蝴蝶，背景是蓝天白云"

❌ **不好的描述**：
- "跳舞"（太简单）
- "猫"（太模糊）

### --num-frames（默认 121）

**视频帧数**，必须是 **16 的倍数 + 1**。

| 帧数 | 时长（24fps） | 建议场景 |
|------|--------------|--------|
| 121 | 5 秒 | 简单动作 |
| 241 | 10 秒 | 中等场景 |
| 361 | 15 秒 | 复杂场景 |

### --frame-rate（默认 24）

**帧率**，视频的流畅度。推荐使用 24。

### --mode

**生成模式**：
- `normal`：普通文本生成视频
- `keyframes`：关键帧动画（需要提供 --image 参数）

### --image

**关键帧图片路径**，用于关键帧模式。可以指定多张图片（至少 2 张）。

## 🔍 理解返回信息

### 成功响应

```json
{
  "success": true,
  "video_id": "vid_abc123",
  "task_id": "task_xyz789",
  "status": "completed",
  "video_url": "https://storage.agnes-ai.com/videos/abc123.mp4",
  "parameters": {
    "num_frames": 121,
    "frame_rate": 24,
    "duration": 5.04,
    "mode": "normal"
  },
  "created_at": "2026-06-25T10:30:00Z"
}
```

### 状态值

| 状态 | 含义 |
|------|------|
| `pending` | 任务已提交，等待处理 |
| `processing` | 正在生成视频 |
| `completed` | 视频生成完成 ✅ |
| `failed` | 生成失败 ❌ |
| `timeout` | 查询超时 |

## 💡 最佳实践

### 1. 详细描述

描述越具体，视频质量越高：

```bash
# ❌ 简单描述
python video_manager.py generate --prompt "跳舞"

# ✅ 详细描述
python video_manager.py generate --prompt "一个穿着白色比基尼的女孩在金色沙滩上随着海浪声跳舞，夕阳西下，海风吹动她的长发"
```

### 2. 合理帧数

根据需要选择合适的帧数：

- **121 帧（5 秒）**：简单动作、特写镜头
- **241 帧（10 秒）**：中等场景、多动作
- **361 帧（15 秒）**：复杂场景、长时间镜头

### 3. 耐心等待

视频生成通常需要 1-5 分钟：

```bash
# 建议使用 --poll 自动查询
python video_manager.py status vid_xxx --poll --interval 5
```

### 4. 检查生成历史

如果视频生成失败，可以查看历史记录：

```bash
python video_manager.py status vid_xxx
```

## 🐛 常见问题

### Q1: 提示 "API Key 未配置"

**解决**：
1. 访问 https://agnes-ai.com 获取 API Key
2. 运行 `agnes key set "您的_API_KEY"`
3. 验证 `agnes key status`

### Q2: 视频生成失败

**可能原因**：
- API Key 无效 → 检查并重新配置
- 网络问题 → 检查网络连接
- 描述违规 → 简化描述
- 服务器繁忙 → 稍后重试

### Q3: 如何加快生成速度？

- 使用较少的帧数（121 vs 361）
- 简化视频描述
- 避免高峰期生成

### Q4: 视频在哪里查看？

生成完成后，会返回 `video_url`，直接在浏览器中打开即可查看。
