# Agnes Video 技能安装指南

## 📦 技能简介

Agnes Video 是一个基于 Agnes AI Video V2.0 模型的 AI 视频生成技能，可以通过文本描述生成高质量视频。

### 核心功能

- ✅ 文本生成视频
- ✅ 关键帧动画
- ✅ 异步任务管理
- ✅ 自动状态查询
- ✅ API Key 配置引导

## 🚀 快速安装

### 方式一：使用技能管理大师（推荐）

如果您已安装"技能管理大师"技能，可以直接让它帮您安装：

```
请帮我全局安装 Agnes Video 技能
```

### 方式二：手动安装

#### 步骤 1：安装 Agnes CLI

```powershell
# 使用 npm 安装（推荐）
npm install -g @agnes-ai/agnes-cli

# 验证安装
agnes --version
```

#### 步骤 2：复制技能文件

```powershell
# 复制整个 agnes-video 目录到全局技能目录
Copy-Item -Path ".trae\skills\agnes-video" -Destination "$env:USERPROFILE\.trae\skills\agnes-video" -Recurse -Force
```

#### 步骤 3：配置 API Key

**方式 A：使用 Agnes CLI（推荐）**

```powershell
# 1. 注册并获取 API Key
agnes key register
# 这会打开浏览器引导您完成注册

# 2. 保存 API Key
agnes key set "您的_API_KEY"

# 3. 验证配置
agnes key status
```

**方式 B：使用环境变量**

```powershell
# 临时设置（当前会话有效）
$env:AGNES_API_KEY = "您的_API_KEY"

# 或永久设置
[System.Environment]::SetEnvironmentVariable(
    "AGNES_API_KEY",
    "您的_API_KEY",
    "User"
)
```

### 步骤 4：验证安装

```powershell
# 进入技能目录
cd "$env:USERPROFILE\.trae\skills\agnes-video"

# 检查系统状态
python video_manager.py check
```

应该看到类似输出：

```
🔍 检查系统配置...

✅ 系统已就绪
   API Key 来源: saved
```

## 📖 使用方法

### 基本命令

```powershell
# 进入技能目录
cd "$env:USERPROFILE\.trae\skills\agnes-video"

# 生成 5 秒视频
python video_manager.py generate `
  --prompt "一只猫在海滩上散步" `
  --num-frames 121 `
  --frame-rate 24

# 查询视频状态
python video_manager.py status <video_id>

# 持续查询直到完成
python video_manager.py status <video_id> --poll --interval 5
```

### 常用参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--prompt` | 视频描述文本 | - |
| `--num-frames` | 视频帧数 | 121 |
| `--frame-rate` | 帧率 (fps) | 24 |
| `--duration` | 视频时长（秒） | 自动计算 |

### 帧数约束

⚠️ **重要**：帧数必须是 **16 的倍数 + 1**

有效值示例：
- 121 帧 ≈ 5 秒 (121 / 24 fps)
- 241 帧 ≈ 10 秒 (241 / 24 fps)
- 361 帧 ≈ 15 秒 (361 / 24 fps)

## ❓ 常见问题

### Q1: 如何获取 API Key？

1. 访问 https://agnes-ai.com
2. 注册账号并登录
3. 在个人中心申请 API Key
4. 使用 `agnes key set "您的_API_KEY"` 保存

### Q2: 为什么视频生成需要等待？

Agnes Video 采用异步生成模式：
1. 提交请求后立即返回 task_id 和 video_id
2. 视频在后台生成（通常需要 1-5 分钟）
3. 使用 video_id 查询生成状态
4. 生成完成后提供视频链接

### Q3: 如何加快生成速度？

- 选择合适的帧数（121 帧通常比 241 帧快）
- 避免高峰期生成
- 使用简单的描述词

### Q4: 生成失败怎么办？

1. 检查 API Key 是否正确
2. 检查网络连接
3. 简化视频描述
4. 减少帧数
5. 查看错误信息并重试

